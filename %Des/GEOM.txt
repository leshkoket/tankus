	Статический класс "Geom"
Описание статического класса: Набор общих функций 2д геометрии.
Члены статического класса:
	+ distanceOf( firstX, firstY, secondX, secondY ) - возвращает расстояние между 2 точками.
	+ radAngleOf( sourceX, sourceY, targetX, targetY ) - возвращает угол от 1 точки до 2й.
	...
	
	
	Класс "Shape"
Описание класса: Абстрактный класс формы.
Члены класса:
	+ new( values:FLOAT[] ) 
	+ values:FLOAT[]() - массив данных определяющих положения формы. Количество и смысл значений определяется в каждом конкретном классе формы.
	+ value:FLOAT(index:INT)/setValue(index:INT, newValue:FLOAT) - получение/установка того или иного значения.
	type:INT() ? - 1 из типов формы, определяемый константой
	
	+ toString() - выдача строки формата value0:value1....
	+ equals( other:Shape, precision:FLOAT ):BOOL - сравнение каждой координаты и типов обеих форм на одинаковость, при заданной точности для значений.

	
Константы класса:
	enum{ SHAPETYPE_RECT, SHAPETYPE_CIRCLE, SHAPETYPE_POINT }.
	SHAPE_VALUE_PRECISION - очень очень очень ОЧЕНЬ маленькое число обратное 1 с нулями и положительное.


	Класс "Point" : "Shape"
Описание класса: Точка с координатами x,y.
Члены класса
	+ new( x,y )
	+ x(), y() - координаты точки (значения #0-#1).
	+ setX(x), setY(y)  - установка координат.
	
	
	Класс "Rect" : "Shape"
Описание класса: прямоугольник с координатами 2 стенок и 2 размеров.
Члены класса:
	+ new( left, top, width, height )
	+@ createFromWalls(left,right, top,bottom) - альтернативный вариант создания прямоугольника по координатам 4 стенок.
	
	+ left(), top(), width(), height() - позиция с левого верха и размеры (значения #0 - #3).
	+ setLeft(value), setTop(value), setWidth(value), setHeight(value) - установка значений.
	
	+ point( proportedX:FLOAT, proportedY:FLOAT ):Point - выдача точки по относительным (как если бы это был квадрат 1:1 размеров в левоверхней позиции 0:0) координатам. Например: point(0,0) - левая верхняя, point(1,1) - правая нижняя, point(0.5,0.5) - центральная...
	+ setPoint(proportedX,proportedY, value) - перемещение прямоугольника за эту точку.
	
	+ corrected:Rect() - выдача нового прямоугольника, так что названия стенок соотв. их положениям, т.е. размеры не отрицательны, притом, что внешне он не поменяется.
	
	+ collidesWithRect( other:Rect ):BOOL - проверка, что оба прямоугольника имеют общие точки на пространстве (сталкиваются).
	
	
	
	Класс "Circle"
Описание класса: круг с 2 координатами центральной позиции и радиусом.
Члены класса:
	+ new( centerX, centerY, radius )
	+ centerX(), centerY(), radius() - значения круга (#0 - #2).
	+ setCenterX(value), setCenterY(value), setRadius(radius) - установка их.
	+ point(proportedX, proportedY) - точка в радиусе через его внешний квадрат по относительным координатам.
	+ setPoint(proportedX, proportedY, value) - перемещение круга за эту точку.
	+ externalQuad():Rect - прямоугольник внешний, в который аккурат входит круг.
	+ internalQuad():Rect - прямоугольник внутренний, который аккурат входит в круг.
	
	+ colidesWithCircle( other:Circle ):BOOL - проверка, что оба круга имеют одинаковые точки на пространстве.
